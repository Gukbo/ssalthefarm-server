# 쌀더팜 (ssalthefarm)

> 게이머들을 위한 정보 공유 쉼터, 쌀더팜

## 🛠 기술 스택 (Tech Stack)
- **Framework:** NestJS
- **Language:** TypeScript
- **Database:** 
- **ORM:** TypeORM

## ✨ 주요 기능 (Key Features)
- **게시글 CRUD:** 게시글 작성, 전체 목록 조회, 상세 조회, 수정, 삭제 기능
- **검색 기능:** Like 연산자를 활용한 제목 키워드 부분 일치 검색
- **유효성 검사:** `class-validator` 및 `ValidationPipe`를 이용한 데이터 무결성 보장
- **CORS 설정:** 프론트엔드 애플리케이션과의 원활한 통신을 위한 설정 완료
